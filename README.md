# Northwest Polishing & Buffing Website

A modern, industrial-themed website for Northwest Polishing & Buffing, built with React, Tailwind CSS, and Framer Motion.

## Project Structure

- `src/components/Layout.tsx`: Main layout including Navigation and Footer.
- `src/pages/Home/index.tsx`: Single-page experience with Hero, Capabilities, Facility, Gallery, and Contact sections.
- `src/index.css`: Global styles including "Industrial Bold" theme (Dark background, Gold accents) and fonts (Bebas Neue, Roboto).

## key Features

- **Industrial Design**: Dark aesthetic with metallic gold highlights matching the industry.
- **Responsive Layout**: Fully functional on mobile, tablet, and desktop.
- **Enhanced Assets**: AI-enhanced imagery for professional presentation.
- **SEO Optimized**: Includes meta tags, sitemap, and structured data for local business.
- **Fast Performance**: Lazy loading and optimized assets.
